# What is this?

Get perfect designs for btn.

#installition

`npm i add-demo-btn-for-project --save`

Then...

```
import {addDemoBtn} from "add-demo-btn-for-project";

addDemoBtn({
    color: "danger_btn"
})

```

## options

We have two options:

- _color_ - danger_btn | success_btn (Defaults to danger_btn)
